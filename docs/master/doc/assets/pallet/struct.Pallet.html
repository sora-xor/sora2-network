pub fn force_mint( origin: OriginFor<T>, asset_id: T::AssetId, to: T::AccountId, amount: Balance ) -> DispatchResultWithPostInfo

Performs an unchecked Asset mint, can only be done by root account.

Should be used as extrinsic call only. Currencies::updated_balance() should be deprecated. Using force_mint allows us to perform extra actions for minting, such as buy-back, extra-minting and etc.

  • origin: caller Account, which issues Asset minting,
  • asset_id: Id of minted Asset,
  • to: Id of Account, to which Asset amount is minted,
  • amount: minted Asset amount.
